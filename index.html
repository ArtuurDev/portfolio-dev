# 💼 Portfólio Dev Projeto desenvolvido como desafio da formação **Full-Stack da Rocketseat**, com o objetivo de criar uma landing page de portfólio moderna, responsiva e visualmente atraente para apresentar projetos, habilidades, serviços e formas de contato. ## ✨ Sobre o Projeto O **Portfólio Dev** simula a página pessoal de um desenvolvedor Full-Stack, reunindo informações profissionais em uma interface organizada e intuitiva. A aplicação foi construída com foco em boas práticas de desenvolvimento front-end, utilizando HTML semântico e CSS moderno para criar uma experiência agradável ao usuário. ## 🚀 Funcionalidades * Apresentação profissional do desenvolvedor * Exibição das principais tecnologias utilizadas * Galeria de projetos em destaque * Seção de serviços oferecidos * Área de contato com redes sociais * Layout responsivo e organizado * Interface moderna inspirada em portfólios profissionais ## 🛠️ Tecnologias Utilizadas * HTML5 * CSS3 * Flexbox * CSS Grid * Variáveis CSS * Google Fonts ## 📂 Estrutura do Projeto ```bash . ├── assets/ │ ├── images/ │ └── icons/ │ ├── styles/ │ ├── global.css │ ├── about.css │ ├── sections.css │ ├── utility.css │ └── index.css │ └── index.html ``` ## 📋 Seções da Página ### 👨‍💻 Sobre Apresentação do desenvolvedor com foto, descrição profissional e principais tecnologias utilizadas. ### 📁 Projetos Exibição de projetos desenvolvidos, contendo imagem, nome e descrição de cada aplicação. ### 🛠️ Serviços Área dedicada aos serviços oferecidos, como desenvolvimento de interfaces, APIs e aplicações web. ### 📞 Contato Links para redes sociais e canais de comunicação profissional. ## 🎯 Conceitos Praticados * HTML Semântico * Estruturação de Landing Pages * Flexbox * CSS Grid * Componentização de Layout * Responsividade * Organização de CSS por módulos * Design Systems * Boas práticas de UI ## 🎓 Aprendizados Durante o desenvolvimento deste projeto foram reforçados conceitos essenciais para a construção de interfaces modernas, incluindo organização de estilos, estruturação semântica, composição de layouts complexos e criação de componentes reutilizáveis. --- Desenvolvido durante a formação **Full-Stack da Rocketseat** 🚀
