# Portfólio Dev

Projeto realizado como desáfio da formação fullstack da rocketseat.

## Estrutura do projeto

- `index.html` — página principal do portfólio
- `styles/` — arquivos de estilos CSS
  - `global.css` — estilos globais e componentes comuns
  - `about.css` — estilos da seção "Sobre"
  - `sections.css` — estilos das seções de projetos, serviços e contato
  - `utility.css` — utilitários de layout e classes de grade/controle
- `assets/` — imagens e ícones usados no projeto

## Seções incluídas

- `#about` — apresentação pessoal com foto, título e resumo
- `#projetos` — portfólio de projetos em destaque
- `#servicos` — serviços oferecidos
- `#contato` — links de contato e redes sociais
